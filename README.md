# README

我并不在意论文的一作归属，当时主动让出署名，是因为这篇论文对我来说并不重要。相比之下，我更需要一个完整的暑假。让我生气的，不是署名，而是整个过程中持续的消耗：

- **缩水严重的摆烂时光，即使抓紧时间，仍然会因为另外两个人的拖延和打扰，迟迟不能真正摆烂**；
- **没有基本“求人办事”的态度**；
- **提供后续实验结果后仍被挑剔格式**；
- **会议被鸽很久，浪费我的休息时间**。

即便如此，我依然没有打算公开这件事。
真正让我愤怒的，是——**他们在没有征得我同意的情况下，将那篇论文直接替代为毕业设计。**
更讽刺的是，在那之前我还没有拉黑他们，他们完全有机会提前沟通。

给到各位本科生/研究生/博士生的建议是，假如你毕业了，有成果没中，但也不想管了，可以让别人接手，但是一旦交接完别人还来烦你，就要马上拉黑！不管导师或对方多“有威严”，只要没有实实在在的好处，就要马上拉黑它！
